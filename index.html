Lazy Neural Network Optimization

This project focused on optimizing deep neural networks by introducing an adaptive early exit mechanism. The method allows intermediate layers to determine whether further computation is necessary based on confidence thresholds, reducing the computational overhead of deep models.

The results showed a significant reduction in inference time while maintaining high accuracy. The project was implemented in TensorFlow and PyTorch and was evaluated on large-scale datasets for various classification tasks.

Benchmarking 1BitLLM Model Using ONNX and IREE Compiler

This project aimed to analyze and optimize the performance of the 1BitLLM model using ONNX and IREE compilers. The model was benchmarked across different hardware configurations, including CPU and GPU, to assess its scalability and latency.

Extensive experiments with varying token sizes and dataset complexities demonstrated that compiler optimizations could enhance model execution speed significantly. The results helped in designing more efficient inference pipelines for large-scale language models.

Snapea - Optimized Neural Network Inference

Snapea is a framework designed to accelerate deep neural network inference by skipping redundant computations dynamically. By leveraging confidence-based skipping mechanisms, the system reduces unnecessary activations and computations in deep models.

Experiments conducted on GoogLeNet, ResNet, and VGG-Net demonstrated a 74% improvement in inference speed without significant loss in accuracy. The project was optimized for deployment on edge devices where computational efficiency is critical.

SimNet - Computer Architecture Simulation

SimNet is a CNN-based simulation framework that predicts computer architecture behaviors, such as branch prediction and fetch latency. The model was trained to simulate microarchitecture behaviors, replacing traditional cycle-accurate simulations.

Through extensive benchmarking, SimNet achieved a 55x improvement in simulation throughput while maintaining low error rates, demonstrating the potential of deep learning in accelerating hardware simulations.

Music Genre Classification Using Machine Learning

The music genre classification project utilized machine learning techniques to categorize songs based on extracted audio features. The dataset included various musical genres, and Mel-frequency cepstral coefficients (MFCCs) were employed for feature extraction.

The trained model was deployed as a web-based tool that allowed users to upload audio files for real-time genre classification. The results showed promising performance in distinguishing between major music genres, showcasing AI's role in music analysis.

Automatic Number Plate Recognition (ANPR) System

The ANPR system leveraged YOLOv5 for object detection and EasyOCR for text extraction, enabling automated number plate recognition from images and video streams. The model was fine-tuned to handle different lighting conditions and occlusions.

With real-time processing capabilities, the system was deployed for traffic monitoring and automated toll collection applications. Post-processing techniques helped improve recognition accuracy in complex environments.

Flight Insurance on Ethereum Blockchain

This project developed a decentralized smart contract system that automatically processes flight insurance claims based on real-time flight data. The contract was implemented on the Ethereum blockchain, ensuring transparency and automation.

By leveraging oracles for fetching flight status data, the system eliminated the need for manual claim processing, reducing delays and operational costs. The project showcased how blockchain can revolutionize the insurance sector.
