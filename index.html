Projects

Lazy Neural Network Optimization

This project explores entropy-based early-exit inference to reduce latency and computational cost in deep neural networks. The architecture is designed to dynamically evaluate confidence at multiple checkpoints, allowing inputs to exit early if prediction confidence crosses a threshold.

Implemented using PyTorch and TensorFlow, LazyNN showed significant improvements in inference efficiency for ResNet-based models on CIFAR datasets, without major compromise in accuracy.

1BitLLM Model Benchmarking

This project benchmarks 1-bit quantized LLMs using ONNX Runtime and the IREE compiler. The primary goal is to evaluate performance across different batch sizes and hardware platforms, focusing on latency, memory usage, and throughput.

Results indicate that 1-bit models significantly reduce memory consumption and can maintain competitive accuracy, especially when compiled with IREE for CPU inference.

Snapea - Efficient Inference

Snapea implements a confidence-aware mechanism that prunes and skips layers in a CNN dynamically. Inspired by energy-aware embedded ML systems, the project aims to balance accuracy and computational load.

We trained the model with adaptive backpropagation logic and simulated real-world edge conditions. This is especially useful in power-constrained scenarios like IoT vision nodes.

SimNet - Architecture Simulation

SimNet is a deep learning-based microarchitectural prediction tool designed to improve the efficiency of CPU pipeline simulations. We use supervised learning on cycle-level trace data to approximate bottleneck identification.

The project helped reduce simulation overhead by over 40% for SPEC workloads when integrated with GEM5 traces.

Music Genre Classification

This system classifies audio tracks into genres using MFCC features and a CNN-based classifier. A Flask web app interface was built to demonstrate real-time audio classification.

We used the GTZAN dataset and implemented both CNN and KNN variants to analyze the trade-off between model complexity and interpretability.

Automatic Number Plate Recognition

A YOLOv5 + EasyOCR pipeline built for real-time traffic monitoring. It detects number plates from live video, extracts alphanumeric text, and saves structured logs.

The system is optimized using OpenCV and NumPy for frame processing and runs efficiently on low-powered GPUs or Jetson devices.

Flight Insurance on Blockchain

This smart contract project automates flight insurance claims on the Ethereum network. It fetches live flight status and pays passengers automatically in case of delays.

Developed using Solidity and Chainlink Oracle, it showcases the intersection of real-world automation and decentralized finance (DeFi).
