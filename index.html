Projects

LEXA – Lightweight RAG with Early Exit

LEXA is a Retrieval-Augmented Generation (RAG) framework tailored for efficient, local deployment on consumer-grade GPUs. It leverages MiniLM for fast document embedding and cosine similarity for retrieval, coupled with a TinyLlama 1.1B model for generation. The architecture is optimized for offline use with no cloud dependency.

The system integrates entropy-based early-exit logic to reduce token generation latency. FastAPI powers the backend, with visual debugging for token-level entropy. Benchmarks focused on latency, energy, and utilization across devices.

Results: 40% latency reduction, 30% energy savings, fully offline and open-source stack.

1BitLLM – Benchmarking and Optimization

Benchmarked 1-bit quantized LLMs using ONNX and IREE. Focused on profiling latency, memory usage, and compiler optimizations for ternary weights. Traced execution with nvprof and IREE tools to evaluate performance on CPUs and GPUs.

Results: 2.5×–3.2× speedup, 60% memory reduction, reproducible compiler benchmark suite.

Snapea – Efficient Inference Framework

Snapea uses entropy-based skipping of CNN layers during inference. The model dynamically exits based on low-uncertainty conditions. Evaluation on CIFAR and TinyImageNet showed significant gains.

Results: 25%–35% latency reduction, <1.5% accuracy drop, real-time on Jetson Nano.

SimNet – Microarchitecture Behavior Simulation

Used LSTM + 1D CNNs to predict IPC, cache behavior, and stalls from instruction traces. Trained on SPEC and gem5 data. Enabled rapid architectural exploration without RTL simulation.

Results: 96.1% correlation, replaced 1,000-cycle simulations with 1ms predictions.
