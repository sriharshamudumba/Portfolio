LEXA: Lightweight Retrieval-Augmented Generation

LEXA is a locally hosted, retrieval-augmented generation (RAG) system optimized for low-latency inference. It leverages MiniLM for embedding queries and documents, retrieves context using cosine similarity, and uses TinyLlama for language generation.

We integrated early-exit logic using entropy filtering, FastAPI for deployment, and visualization tools for debugging. LEXA runs entirely offline and demonstrates high efficiency in constrained environments.

Results: Achieved 40% reduction in token generation time and 30% GPU energy savings across multiple QA tasks.

1BitLLM Benchmarking

This project benchmarks LLMs with 1-bit quantized weights. We evaluated inference performance using ONNX Runtime and the IREE compiler, comparing single- and multi-threaded CPU execution across various batch sizes and token lengths.

It showcases the trade-offs between model compression and accuracy while optimizing for real-world latency and memory footprint.

Results: Reduced memory usage by 4× and achieved 1.8× speedup on average across tested scenarios.

Snapea - Efficient CNN Inference

Snapea introduces confidence-based skipping and pruning for CNNs used in edge inference. Each layer dynamically decides to continue or exit based on entropy of predictions, reducing compute cost per image.

The CNN backbone was trained with knowledge distillation and exit-aware loss to preserve accuracy at intermediate layers.

Results: Reduced inference time by 35% and maintained 91%+ accuracy on CIFAR-10 even when exiting early at 2 of 4 exits.

PyTorchResNetEntropy-Based ExitKnowledge Distillation

SimNet - Architecture Simulation Prediction

SimNet is a supervised learning model to estimate microarchitectural simulation outcomes. It replaces expensive trace-based simulators with learned approximators.

Trained on GEM5 traces, the model predicts cache hits, pipeline stalls, and IPC using input features from configuration files and early trace snippets.

Results: Delivered 2.3× simulation speedup with over 85% accuracy on SPEC benchmarks.

Music Genre Classification

This project classifies audio files into musical genres using CNNs on MFCC features. We combined GTZAN and FMA datasets, applying augmentations such as pitch shift and time stretch to improve generalization.

The final model was served as a Flask web app where users could upload tracks and receive genre predictions in real time.

Results: Achieved 93.4% accuracy; reduced genre confusion between Rock/Pop using confidence thresholding; built mobile-friendly UI for testing.

Automatic Number Plate Recognition (ANPR)

This real-time number plate recognition system uses YOLOv5 for plate detection and EasyOCR for text extraction. Frames are processed using OpenCV and inference runs on Jetson-compatible low-power devices.

We logged plates, timestamps, and vehicle types to structured CSV files for downstream traffic analysis.

Results: Maintained 98% detection accuracy with <100ms latency/frame on GTX 1050Ti; deployed to edge surveillance hardware.
