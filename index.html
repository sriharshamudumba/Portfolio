Projects

Lazy Neural Network Optimization

What is it? This project focuses on optimizing neural network inference by implementing adaptive early exit mechanisms. By analyzing intermediate activations, we can reduce redundant computations, improving efficiency without sacrificing accuracy.

What’s happening? The model monitors confidence scores during inference and dynamically decides whether to exit early or continue processing, reducing latency and computational cost.

GitHub Repository: Lazy NN Optimization

Benchmarking 1BitLLM

What is it? This project analyzes the performance of 1BitLLM models under different computing environments using ONNX and IREE compilers.

What’s happening? The study evaluates latency, scalability, and throughput to optimize deployment efficiency for AI workloads in cloud and edge computing.

GitHub Repository: 1BitLLM Benchmarking

SnaPEA Inference Optimization

What is it? SnaPEA enhances deep learning model efficiency by dynamically pruning non-contributive activations using percentile-based thresholds.

What’s happening? The framework integrates sparse computation, mixed precision training, and early exit mechanisms to accelerate inference by up to 74% with minimal accuracy loss.

GitHub Repository: SnaPEA Inference Optimization

Music Genre Classification

What is it? A convolutional neural network (CNN)-based classifier trained on GTZAN and FMA datasets to predict music genres.

What’s happening? The model extracts low-level audio features and applies deep learning-based classification techniques to achieve high accuracy on real-world music data.

GitHub Repository: Music Genre Classification
